# ☕ Coffee Shop
 
Uma aplicação web para uma cafeteria moderna, com cardápio interativo, sistema de pedidos e uma experiência visual aconchegante para os clientes.
 
---
 
## 🚀 Tecnologias
 
- **HTML5**
- **CSS3**
- **Git** & **GitHub**
---
 
## ✨ Funcionalidades
 
- 📋 Cardápio com categorias (cafés, bebidas, doces, salgados)
- 🛒 Carrinho de pedidos
- 📱 Layout responsivo para mobile e desktop
- 🎨 Design aconchegante inspirado em cafeterias artesanais
---
